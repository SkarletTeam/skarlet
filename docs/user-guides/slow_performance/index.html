Slow Performance

Symptoms:

  • Applications take a long time to open.
  • The computer freezes or lags during tasks.

How to Resolve:

  1. Check for background processes:

  2. Press Ctrl + Shift + Esc to open Task Manager.

  3. Look for programs using high CPU, memory, or disk.

  4. Right-click and end unnecessary tasks.

  5. Disable startup programs:

  6. Open Task Manager, click the Startup tab.
  7. Disable programs that you donâ€™t need running at startup.
  8. Clear temporary files:
  9. Press Windows + R, type temp, and hit Enter.
  10. Delete all files in the folder.
  11. Run Disk Cleanup:
  12. Type Disk Cleanup in the Start menu, select your drive (usually C:), and clean unnecessary files.
  13. Check for malware:
  14. Run a full scan with your antivirus software.
  15. Consider upgrading hardware:
  16. Increase RAM or replace a mechanical hard drive with an SSD.
