Blue Screen of Death (BSOD)

Symptoms:

  • Your computer suddenly crashes and shows a blue screen with an error message.

How to Resolve:

  1. Note the error code:
  2. Write down the error code displayed on the blue screen, like CRITICAL_PROCESS_DIED or MEMORY_MANAGEMENT.
  3. Update drivers:
  4. Open Device Manager and update all drivers, especially for graphics cards, sound cards, and network adapters.
  5. Check for hardware issues:
  6. Run Windows Memory Diagnostic (type it in the search bar) to check for RAM issues.
  7. Use chkdsk to check your hard drive for errors:
    • Open Command Prompt as administrator.
    • Type chkdsk /f /r and press Enter.
  8. Revert recent changes:
  9. If the BSOD started after a recent driver or software installation, roll back to the previous version or uninstall the new software.
  10. System Restore:
  11. If the problem persists, use System Restore to revert your system to a previous working state.
